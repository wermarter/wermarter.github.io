A résumé is a snapshot. A website can be a working notebook. Structured experience data can become a CV, a JSON document, or a new presentation later, while Markdown writing remains easy to own and move.

## Static by default

Most of this page is plain HTML. JavaScript appears only where it adds something useful. This small button is one of those deliberately isolated pieces:
